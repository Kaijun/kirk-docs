## 3.4.4 远程终端

类似 `docker exec` 命令功能，以Terminal形式登入进容器内部。

!> **注意：** 如果在使用远程终端时遇到不能输入或显示中文的情况，请尝试以下解决方法：<br>
1.在您的 Dockerfile 中加入环境变量：`ENV LANG=C.UTF-8`<br>
2.在服务设置 -> 高级配置 -> 环境变量 -> 添加环境变量 -> 变量名为 `LANG`，变量值为 `C.UTF-8`<br>
